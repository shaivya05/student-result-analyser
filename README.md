# Student Result Analyzer

A Python and Streamlit based dashboard to analyze student results.

## Features
- Class statistics — average, highest score, pass/fail count
- Full class report with grades and rankings
- Grade distribution and pass/fail pie charts
- Download report as CSV

## Tech Stack
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run
pip install pandas matplotlib streamlit
streamlit run app.py
